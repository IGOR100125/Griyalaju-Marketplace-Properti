# 🏡 GriyaLaju Property Dataset Analysis

Analisis data properti menggunakan **Python**, **MySQL**, dan teknik **Exploratory Data Analysis (EDA)** untuk memperoleh insight mengenai tren pasar properti, performa agen, transaksi penjualan, serta karakteristik properti.

Repository ini dibuat sebagai portofolio Data Analyst yang berfokus pada proses **data cleaning**, **data exploration**, **SQL analysis**, dan **visualisasi data**.

---

## 📌 Project Overview

Dalam industri properti, data transaksi dan listing dapat dimanfaatkan untuk menghasilkan berbagai insight yang berguna bagi perusahaan maupun agen properti.

Project ini bertujuan untuk:

* Membersihkan dan mempersiapkan dataset.
* Melakukan Exploratory Data Analysis (EDA).
* Menganalisis performa agen properti.
* Menganalisis tren transaksi.
* Mengidentifikasi kota dan tipe properti yang paling diminati.
* Menghasilkan visualisasi yang mudah dipahami.

---

## 🎯 Objectives

* Memahami karakteristik dataset properti.
* Menemukan pola penjualan properti.
* Mengidentifikasi agen dengan performa terbaik.
* Mengetahui wilayah dengan aktivitas transaksi tertinggi.
* Menganalisis harga properti berdasarkan lokasi dan tipe.
* Menyajikan insight yang dapat digunakan sebagai dasar pengambilan keputusan.

## 📊 Analysis Performed

Beberapa analisis yang dilakukan pada project ini meliputi:

### Data Cleaning

* Menghapus data duplikat

### Exploratory Data Analysis (EDA)

* Distribusi harga properti

Contoh analisis menggunakan SQL:

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/IGOR100125/griyalaju-property-dataset-analysis.git
```

Masuk ke folder project

```bash
cd griyalaju-property-dataset-analysis
```

Install dependency

```bash
pip install -r requirements.txt
```

Jalankan Jupyter Notebook

```bash
jupyter notebook
```
